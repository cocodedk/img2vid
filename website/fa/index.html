img2vid

پوشه‌های تصویر مرتب شده را با جلوه محو تدریجی به ویدیوی MP4 تبدیل کنید. رایگان، متن‌باز، و قابل استفاده از طریق CLI یا مرورگر.

دانلود آخرین نسخه

ویژگی‌ها

  • جلوه محو تدریجی — انتقال روان بین تصاویر
  • پشتیبانی از صدا — افزودن موسیقی پس‌زمینه
  • CLI + رابط وب — استفاده از ترمینال یا مرورگر
  • خروجی MP4 — ویدیوی H.264 سازگار با همه دستگاه‌ها
  • مبتنی بر Python — ساخته شده با moviepy

شروع سریع

pip install uv
uv sync
uv run img2vid --help
