img2vid

Convert ordered image folders into crossfaded MP4 slideshows with optional audio soundtrack. Free, open-source, and works from CLI or web UI.

Download Latest

Features

  • Crossfade transitions — Smooth fade between images
  • Audio support — Add a soundtrack to your slideshow
  • CLI + Web UI — Use from terminal or browser
  • Ordered output — Images sorted alphabetically for consistent results
  • MP4 output — Standard H.264 video compatible everywhere
  • Python-native — Built on moviepy, no proprietary tools needed

Quick Start

pip install uv
uv sync
uv run img2vid --help
